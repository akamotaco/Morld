# 시나리오 01: 인트로

## 개요

- **이름**: 인트로
- **설명**: 기본 시나리오 - 마을에서의 시작
- **버전**: 1.0.0

플레이어가 낯선 마을에서 깨어나 기억을 되찾아가는 도입부 시나리오입니다.

---

## 스토리 개요

1. **기상**: 낯선 집에서 깨어남
2. **기억 상실**: 자신이 누구인지 기억나지 않음
3. **직업 선택**: 검사/마법사/도적 중 선택
4. **초기 장비 획득**: 선택한 직업에 맞는 장비 지급
5. **탐색 시작**: 마을을 돌아다니며 정보 수집

---

## 테스트 체크리스트

### 1. 게임 시작 이벤트
- [ ] 게임 시작 시 인트로 모놀로그 표시
- [ ] 3페이지 인트로 텍스트 확인
- [ ] 직업 선택 화면으로 자연스럽게 연결

### 2. 직업 선택 시스템
- [ ] 3가지 직업 선택지 표시 (검술/마법/은밀)
- [ ] 선택 시 YesNo 확인 다이얼로그 표시
- [ ] "아니오" 선택 시 이전 선택 화면으로 복귀
- [ ] "예" 선택 시 직업 확정 및 장비 지급

### 3. 초기 장비 지급
검사 선택 시:
- [ ] 철검 x1
- [ ] 나무 방패 x1

- [ ] 철 투구 x1
- [ ] 체력 포션 x3
- [ ] 마른 빵 x5
- [ ] 물통 x1
- [ ] 동전 주머니 x1

마법사 선택 시:
- [ ] 참나무 지팡이 x1
- [ ] 마법사 로브 x1
- [ ] 마법서: 불꽃 x1
- [ ] 마나 포션 x5
- [ ] 체력 포션 x2
- [ ] 마른 빵 x3
- [ ] 물통 x1
- [ ] 동전 주머니 x1

도적 선택 시:
- [ ] 단검 x1
- [ ] 경장갑 x1
- [ ] 그림자 망토 x1
- [ ] 자물쇠 따개 x3
- [ ] 투척용 나이프 x5
- [ ] 체력 포션 x2
- [ ] 마른 빵 x3
- [ ] 물통 x1
- [ ] 동전 주머니 x1

### 4. NPC 대화 시스템
- [ ] 철수(id:1) 대화 - activity에 따른 대사 변화
- [ ] 영희(id:2) 대화 - 상점 NPC, 영업 중 특수 대사
- [ ] 민수(id:3) 대화 - 과묵한 캐릭터 대사

### 5. 오브젝트 상호작용
- [ ] 거울 보기 - 모놀로그 표시 후 복귀

### 6. 시간 시스템
- [ ] 시간 경과에 따른 NPC activity 변화
- [ ] 시간대별 장소 묘사 변화

### 7. 이동 시스템
- [ ] 장소 간 이동
- [ ] 이동 시간 소요
- [ ] 경로 조건 (열쇠 필요 등)
