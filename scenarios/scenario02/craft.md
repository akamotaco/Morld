**획득 순서:**

2. [ ] 도구함 클릭

## 2. 벌목 테스트

### 사전 준비
- [ ] 도끼 소지 확인 (`axe`)
- [ ] 도끼 장착 (`equip@inventory`) - can:chop 활성화

### 테스트 순서
1. [ ] 숲 깊은 곳(21)으로 이동
2. [ ] 나무(Tree) 클릭
3. [ ] "벌목" 액션이 표시되는지 확인 (can:chop 필요)
4. [ ] 벌목 실행 → 5분 경과
5. [ ] 통나무 획득 확인 (인벤토리)
6. [ ] 나뭇가지 획득 확인 (인벤토리, 선택적)

### 예상 결과
- 통나무(log) 1개 획득
- 나뭇가지(branch) 0~1개 획득 (확률)
- 나무 자원 개수 감소

---

## 3. 통나무 가공 테스트

### 사전 준비
- [ ] 도끼 소지 확인 (`axe`) - passive_props로 can:process 활성화
- [ ] 통나무(log) 소지 확인

### 테스트 순서
1. [ ] 인벤토리에서 통나무 클릭
2. [ ] "나무판으로 가공" 액션이 표시되는지 확인
   - 도끼가 없으면 이 액션이 표시되지 않아야 함
3. [ ] 가공 실행 → 10분 경과
4. [ ] 나무판 3개 획득 확인
5. [ ] 통나무 1개 소비 확인

### 예상 결과
- 통나무 1개 → 나무판(plank) 3개
- 10분 경과

## 4. 플레이어 제작 테스트 (화살)

### 레시피
| 결과물 | 재료 | 제작 시간 | 생성 개수 |
|-------|------|----------|----------|
| 화살 | 나무판 1 + 깃털 1 | 10분 | 5개 |

### 사전 준비
- [ ] 나무판(plank) 1개 이상 소지
- [ ] 깃털(feather) 1개 이상 소지

### 깃털 획득 방법
**참고:** 현재 깃털은 게임 내에서 직접 획득 방법이 구현되지 않음
- 향후: 사냥으로 획득 예정

### 테스트 순서
1. [ ] 화면 하단 "제작" 버튼 클릭 (플레이어 액션)
2. [ ] 카테고리 선택 → "무기" 클릭
3. [ ] "화살" 레시피 클릭
4. [ ] 재료 확인 화면 표시
   - 나무판: 보유/필요 개수 + 체크 표시
   - 깃털: 보유/필요 개수 + 체크 표시
5. [ ] 재료가 충분하면 "제작하기" 버튼 활성화
6. [ ] 재료가 부족하면 "재료가 부족합니다" 표시
7. [ ] 제작 실행 → 10분 경과
8. [ ] 화살 5개 획득 확인
9. [ ] 나무판 1개, 깃털 1개 소비 확인

### 예상 결과
- "화살 5개를 만들었다!" 메시지
- 화살(arrow) 5개 인벤토리에 추가

## 5. 제작대 제작 테스트 (사냥용 활)

### 레시피
| 결과물 | 재료 | 제작 시간 | 생성 개수 |
|-------|------|----------|----------|
| 사냥용 활 | 나무판 2 + 끈 1 | 30분 | 1개 |

### 사전 준비
- [ ] 나무판(plank) 2개 이상 소지
- [ ] 끈(cord) 1개 이상 소지
- [ ] 창고(5)로 이동

### 끈 획득 방법
**참고:** 현재 끈은 게임 내에서 직접 획득 방법이 구현되지 않음
- 향후: 섬유 가공 또는 상점 구매 예정

### 테스트 순서
1. [ ] 창고(5)로 이동
2. [ ] 제작대 클릭
3. [ ] "제작하기" 액션 클릭
4. [ ] 카테고리 선택 → "무기" 클릭
5. [ ] "사냥용 활" 레시피 클릭
6. [ ] 재료 확인 화면 표시
   - 나무판: 보유/필요 개수 + 체크 표시
   - 끈: 보유/필요 개수 + 체크 표시
7. [ ] 재료가 충분하면 "제작하기" 버튼 활성화
8. [ ] 제작 실행 → 30분 경과
9. [ ] 사냥용 활 1개 획득 확인
10. [ ] 나무판 2개, 끈 1개 소비 확인

### 예상 결과
- "사냥용 활을(를) 만들었다!" 메시지
- 사냥용 활(hunting_bow) 1개 인벤토리에 추가

## 6. 제작 분리 테스트

### 플레이어 제작에서 활이 없는지 확인
1. [ ] 아무 곳에서나 "제작" 버튼 클릭
2. [ ] "무기" 카테고리 선택
3. [ ] 화살만 표시되고, 사냥용 활은 표시되지 않아야 함

### 제작대에서 화살이 없는지 확인
1. [ ] 창고(5)로 이동
2. [ ] 제작대 클릭 → "제작하기"
3. [ ] "무기" 카테고리 선택
4. [ ] 사냥용 활만 표시되고, 화살은 표시되지 않아야 함

## 참고: 아이템 unique_id 목록

### 도구
| unique_id | 이름 | 위치 | 기능 |
|-----------|------|------|------|
| `axe` | 도끼 | 창고 도구함 | 장착: can:chop, 보유: can:process |
| `fishing_rod` | 낚시대 | 창고 도구함 | 장착: can:fish |

### 재료
| unique_id | 이름 | 카테고리 | 획득 방법 |
|-----------|------|---------|----------|
| `log` | 통나무 | material | 벌목 (도끼 장착) |
| `branch` | 나뭇가지 | material | 벌목 시 추가 획득 |
| `plank` | 나무판 | material | 통나무 가공 (도끼 보유) |
| `cord` | 끈 | material | (미구현) |
| `feather` | 깃털 | material | (미구현) |

### 제작품
| unique_id | 이름 | 제작 위치 |
|-----------|------|----------|
| `hunting_bow` | 사냥용 활 | 제작대 |
| `arrow` | 화살 | 플레이어 제작 |

### 네이밍 규칙
- 재료류: category = "material"
- 도구류: equip_props에 특수 능력 부여
