## 1. 낚시 테스트

### 낚시대 획득 방법
| 위치 | Location ID | 오브젝트 |
|------|------------|---------|
| 창고 | 5 | 도구함(Toolbox) 안에 낚시대 있음 |

**이동 경로:**
```
거실(1) → 2층 복도(14) → 창고(5)
```

**획득 순서:**
1. [ ] 창고(5)로 이동
2. [ ] 도구함 클릭
3. [ ] 낚시대 "가져가기"
4. [ ] 인벤토리에서 낚시대 확인

### 사전 준비
- [ ] 낚시대 소지 확인 (`fishing_rod`)
- [ ] 낚시대 장착 (`equip@inventory`)

### 테스트 순서
1. [ ] 강가(22)로 이동
2. [ ] 낚시터(FishingSpot) 클릭
3. [ ] "낚시" 액션이 표시되는지 확인 (can:fish 필요)
4. [ ] 낚시 실행 → 15분 경과
5. [ ] 70% 확률로 생선(food_fish) 획득 확인
6. [ ] 인벤토리에서 생선 확인

### 예상 결과
- 성공: "물고기를 잡았다!" + 생선 1개 획득
- 실패: "아무것도 잡히지 않았다."

## 2. 채집 테스트

### 채집 가능 자원

| 오브젝트 | 위치 | 생성 아이템 | 초기 개수 |
|---------|------|-----------|----------|
| 사과나무 | 숲 깊은 곳(21) | 사과 | 2개 |
| 산딸기 덤불 | 채집터(23), 강가(22) | 산딸기 | 3개, 2개 |
| 버섯 군락 | 채집터(23) | 버섯 | 2개 |

### 테스트 순서 (채집터 기준)
1. [ ] 채집터(23)로 이동
2. [ ] 산딸기 덤불 클릭 → container 액션으로 인벤토리 표시
3. [ ] 산딸기 "가져가기" 실행
4. [ ] 인벤토리에서 산딸기(food_wild_berry) 확인
5. [ ] 버섯 군락에서도 동일하게 버섯(food_mushroom) 채집

### 테스트 순서 (숲 깊은 곳 기준)
1. [ ] 숲 깊은 곳(21)으로 이동
2. [ ] 사과나무 클릭
3. [ ] 사과(food_apple) 채집

## 3. 조리 테스트

### 현재 레시피

| 결과물 | 재료 | 조리시간 |
|-------|------|---------|
| 구운 고기 | 고기 1 | 15분 |
| 허브 샐러드 | 약초 2 | 5분 |
| 고기 스튜 | 고기 1 + 약초 1 | 30분 |
| 빵 | 밀가루 1 + 물 1 | 20분 |

### 사전 준비
- [ ] 조리 재료 확보 (테스트용으로 약초 2개 또는 고기 1개)
- [ ] 주방(2)으로 이동

### 테스트 순서
1. [ ] 아궁이(Stove) 클릭
2. [ ] "재료 확인" 액션으로 현재 재료 확인
3. [ ] 인벤토리에서 재료 선택 → "넣기" 액션
4. [ ] 다시 아궁이 클릭 → "조리하기" 액션
5. [ ] 레시피 매칭 확인:
   - 재료 정확히 일치해야 함 (예: 약초 2개만 넣어야 허브 샐러드)
   - 재료가 맞지 않으면 "이 재료로는 만들 수 있는 것이 없다"
6. [ ] 조리 완료 후 결과물 확인
7. [ ] 결과물 "가져가기"로 인벤토리에 추가
