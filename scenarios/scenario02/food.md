## 1. 낚시 테스트

### 낚시대 획득 방법
| 위치 | Location ID | 오브젝트 |
|------|------------|---------|
| 창고 | 5 | 도구함(Toolbox) 안에 낚시대 있음 |

**이동 경로:**
```
거실(1) → 2층 복도(14) → 창고(5)
```

**획득 순서:**
1. [x] 창고(5)로 이동
2. [x] 도구함 클릭
3. [x] 낚시대 "가져가기"
4. [x] 인벤토리에서 낚시대 확인

### 사전 준비
- [x] 낚시대 소지 확인 (`fishing_rod`)
- [x] 낚시대 장착 (`equip@inventory`)

### 테스트 순서
1. [x] 강가(22)로 이동
2. [x] 낚시터(FishingSpot) 클릭
3. [x] "낚시" 액션이 표시되는지 확인 (can:fish 필요)
4. [x] 낚시 실행 → 15분 경과
5. [x] 70% 확률로 생선(food_fish) 획득 확인
6. [x] 인벤토리에서 생선 확인

### 예상 결과
- 성공: "물고기를 잡았다!" + 생선 1개 획득
- 실패: "아무것도 잡히지 않았다."

## 3. 조리 테스트

### 현재 레시피

| 결과물 | 재료 | 도구 | 조리시간 | 포만감 |
|-------|------|------|---------|-------|
| 구운 생선 | 생선 1 | 아궁이 | 10분 | 35 |
| 버섯 스튜 | 버섯 2 | 아궁이 | 20분 | 45 |
| 과일 샐러드 | 사과 1 + 산딸기 2 | 아궁이 | 5분 | 40 |
| 허브티 | 약초 1 | 주전자 | 5분 | 15 |

### 재료 카테고리
| 재료 | 카테고리 | 획득 위치 | 방법 |
|-----|---------|---------|------|
| 생선 | food_ingredient | 강가(22) | 낚시 (낚시대 장착 필요) |
| 버섯 | food_ingredient | 채집터(23) | 버섯 군락에서 채집 |
| 사과 | food_ingredient | 숲 깊은 곳(21) | 사과나무에서 채집 |
| 산딸기 | food_ingredient | 채집터(23), 강가(22) | 산딸기 덤불에서 채집 |
| 약초 | drink_ingredient | 뒷마당(13) | 약초밭에서 채집 |

### 사전 준비
- [ ] 조리 재료 확보
- [ ] 주방(2)으로 이동

### 테스트 순서 (허브티 예시 - 주전자 사용)
1. [ ] 뒷마당에서 약초 1개 채집
2. [ ] 주방(2)으로 이동
3. [ ] 주전자(Kettle) 클릭
4. [ ] "재료 넣기" 클릭 → 약초 선택
   - 약초는 drink_ingredient이므로 주전자에만 넣을 수 있음
   - 아궁이에는 약초를 넣을 수 없음 (food_ingredient만 가능)
5. [ ] 다시 주전자 클릭 → "끓이기" 액션
6. [ ] 레시피 매칭 → 허브티 완성 (5분 경과)
7. [ ] 주전자 클릭 → container UI로 허브티 확인
8. [ ] 허브티 "가져가기"로 인벤토리에 추가

### 테스트 순서 (과일 샐러드 예시 - 아궁이 사용)
1. [ ] 숲 깊은 곳에서 사과 1개 채집
2. [ ] 채집터에서 산딸기 2개 채집
3. [ ] 주방(2)으로 이동
4. [ ] 아궁이(Stove) 클릭
5. [ ] "재료 넣기" 클릭 → 사과 선택
6. [ ] 다시 "재료 넣기" → 산딸기 2개 넣기
7. [ ] "조리하기" → 과일 샐러드 완성 (5분 경과)
8. [ ] 아궁이 container UI에서 과일 샐러드 "가져가기"

## 참고: 아이템 unique_id 목록

### 채집 재료
| unique_id | 이름 | 카테고리 |
|-----------|------|---------|
| `food_wild_berry` | 산딸기 | food_ingredient |
| `food_apple` | 사과 | food_ingredient |
| `food_mushroom` | 버섯 | food_ingredient |
| `food_fish` | 생선 | food_ingredient |
| `food_herb` | 약초 | drink_ingredient |

### 조리 음식
- `food_cooked_fish` - 구운 생선
- `food_mushroom_stew` - 버섯 스튜
- `food_fruit_salad` - 과일 샐러드
- `drink_herb_tea` - 허브티

### 네이밍 규칙
- 음식류: `food_` 접두사
- 음료류: `drink_` 접두사

### 카테고리 시스템
- `food_ingredient` - 음식 재료 (아궁이에만 넣을 수 있음)
- `drink_ingredient` - 음료 재료 (주전자에만 넣을 수 있음)
