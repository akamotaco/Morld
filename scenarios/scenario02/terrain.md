3. [ ] 숲 깊은 곳(21)으로 이동 → 나무 오브젝트 확인

5. [ ] 채집터(23)로 이동 → 자원 오브젝트 확인
6. [ ] 숲 입구(20)로 복귀
7. [ ] 앞마당(12)으로 복귀 → 10분 경과 확인

| 숲 깊은 곳(21) | 나무(Tree) | 통나무 3개 | 24시간/개 |

### 테스트 순서

6. [ ] 통나무 획득 확인

2. [ ] 24시간(게임 내) 경과
