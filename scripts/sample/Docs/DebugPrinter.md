## 사용법

```csharp
using PathFinding.Serialization;

// 콘솔에 직접 출력
DebugPrinter.DumpWorld(world);
DebugPrinter.DumpNPCs(gameWorld);
DebugPrinter.DumpGameWorld(gameWorld, detailed: true);
DebugPrinter.DumpSnapshot(gameWorld);

// 문자열로 받기
string worldSummary = DebugPrinter.PrintWorld(world);
string npcSummary = DebugPrinter.PrintNPCs(gameWorld);
string graph = DebugPrinter.PrintWorldGraph(world);

// 단일 NPC
string npcDetail = DebugPrinter.PrintNPC(npc, world);
string npcStatus = DebugPrinter.PrintNPCStatus(npc, world);  // 한 줄
```

## 실행 중 단축키

| 키 | 기능 |
|----|------|
| `W` | World 정보 출력 |
| `N` | NPC 정보 출력 |
| `D` | 전체 상세 정보 출력 |
| `G` | World 그래프 출력 |
| `S` | 분리 저장 |
| `A` | 통합 저장 |
